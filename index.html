The primary objective was to facilitate the assignment of users to customers through APIs, ensuring seamless data flow and efficient system integration. Given the scale and complexity of customer data, the project required robust automation and integration with cloud services to enhance accuracy and efficiency.




    ED Retirement

    ED-Retirement project was aimed at modernizing and replacing the existing data flow with enhanced business rules across multiple data pathways. The project focused on improving the efficiency of asset onboarding, customer and dealer attachment, subscription management, and IoT gateway registration through API-driven automation. The objective was to ensure seamless integration of these processes while maintaining data accuracy and optimizing system performance.


    Parts Cat Com

    CUOB UI project was focused on front-end UI testing to ensure a seamless user experience in applications like Parts Cat Com and Vision Link. The project aimed to validate user access flows, including creating users through email invitations or direct user requests from the platforms. Additionally, results were monitored and reported using CloudWatch and integrated into ADO Test Results for tracking



    Foresight

    Foresight project is designed to collect, analyze, and provide insights on fault data, fluid data, and work data from assets enrolled in the Foresight program. By leveraging data analytics, the project aims to enhance predictive maintenance and optimize asset performance. The system processes large volumes of transit data, generating actionable recommendations for users based on real-time insights.
